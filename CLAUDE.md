# TASheet

三角机构 TRPG 角色卡生成器。Vite + React + TypeScript + Tailwind CSS，部署于 Cloudflare Pages。

## 环境

- **OS**: Windows 11
- **Shell**: PowerShell 或 cmd（不要用 Linux bash 语法）
- **Node**: `D:\Nodejs\node.exe`
- **npm**: `D:\Nodejs\npm.cmd`
- **Git**: `D:\git\Git\cmd\git.exe`

常用命令：
```powershell
npm run dev        # 启动开发服务器
npm run build      # 构建到 dist\
npm run preview    # 预览构建产物
```
